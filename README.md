# Sprint_7
Proyecto del sprint 7. Desarrollo de Software
Este proyecto consta de demostrar los conocimientos adquiridos durante el sprint 7, en el cuál usaremos git y github para tener un control de versiones, despues el uso de ploty.express para hacer diagramas de manera sencilla y visualizarlos en un archivo de Jupyter Notebook, para después realizar una aplicación web con el modulo de streamlit y una vez comprobado subir el archivo a render.
